# TP1 – Display Your Name on LCD Using ESP32 (Wokwi Simulation)

##  Objective

In this practical lab, students will:

* Build a virtual hardware system using **Wokwi**
* Add and connect:

  * ESP32 board
  * LCD 16x2 (I2C)
  * Required wiring
* Write MicroPython code
* Display their **name on the LCD screen**


# Required Components (Student Task)

Students must:

1. Add **ESP32 board**
2. Add **LCD1602 I2C**
3. Add **Breadboard** 
4. Connect components correctly
5. Configure I2C wiring
6. Write the MicroPython program
7. Run the simulation

# Software Requirements

Students must write a MicroPython program that:

1. Connects to I2C
2. Initializes the LCD
3. Clears the screen
4. Displays their full name

Example Output:

```
John Smith
TP1 IoT
```
